- glues everything together in the document

Hyperlinks - make web connected

Major Pieces of the Web

- Your Computer Browser
- The Internet
- HTTP - language used to send HTML documents
- Servers

HTML consists of

- text content
- markup
- references to other documents
- links to other pages

HTML Markup

- < b>BOLD < /b>
- < em>italics < /em>

HTML Attributes

- < tag ATTR="value"> contents < /tag>
tags can have multiple attributes

- Anchor < a> is used to make links
< a href - "www.reddit.com"> derp < /a>

Images

< img> is used to insert images
< img SRC="URL" alt="text">
alt inserts text in place of image in case image fails to appear
< img> doesn't have a closing tag

White space

Multiple spaces or enters are reduced to one space
- Inline elements influence a line
- Container elements make an invisble box with height and width
and do nothing but contain stuff inside themselves.
< p> is a container element, while < br> is an inline element

HTML Documents

< !DOCTYPE HTML> - doctype in html 5
< html> document < /html>
< head> may contain metadata, javascript, css,
or < title> text < /title> < /head>

CSS is a styling language

Summary

A web page is a text document written in HTML. Web Browsers
read these documents, then interpret and display them.

An HTML element refers to everything within a set of opening and closing tags

An attribute is a property of an HTML element



Creating a Structured Document

Tree-Like Structure

HTML is set up in tree-like structure
with each part having sub-parts that branch
off to the final elements.
- HTML and CSS are languages
- HTML controls the structure of the page
tags define an element in the tree
- CSS controls the style by using the HTML language
- DOM = "Document Object Model" = treelike structure

Boxing a Design

- Everything on a webpage is actually a box.
- There is a grid to align the boxes
- To boxify a design is to subdivide the document
into sections for elements by using boxes
< div> define boxes < /div>
- Use class to give boxes names
< div class="box 1"> text < /div>

Adding CSS Style and HTML Structure

< style>style < /style> can be used in html
< link> is used to reference style code in CSS

Cascading Style Sheets (CSS)

Using CSS helps avoid repetition by applying style
to similar elements. This saves the programmer
extra work and their employer money.

Inheritance - properties are applied not only to elements,
but also it's decendents on the document tree.

Text-related properties are inherited,
while box-related properties are not inherited.

When more than one rule applies, the more specific
rule takes presidence.

Selectors

A selector defines elements that a style will apply to.

Class selectors add style only to certains elements.

declaration = property:value;

Comments

html comments are placed in < ! text>
CSS comments are placed in /*text*/

The Box Model

A box consists of four parts each inside the other.
The margin is on the outside and transparent.
The border surrounds the padding with the
content in the middle that provides the
protective space for the image or text.

When using *{box-sizing:border-box;}, browser
specific prefixes may be needed for older browsers.

Browsers uses a default style for elements in the
absense of a coded style

Code, Test, Refine

While working from the biggest to the smallest,
boxify, tag, size and position boxes, and write html.
Repeat the process until the webpage
matches the design mockup.
